# Learning Objectives
- Elements have these properties within the box model
- (\[{content}]) {padding} \[border] margin() 
- Classes are reusable but IDs are unique


```css
*{
	margin: 0;
	padding: 0;
	box-sizing: border-box;
}
body {
	margin: 2rem;
	background-color: #fffdb7;
	color: black;
	font-size: 1.5rem;
}
h1 {
	margin-bottom: 2rem;
}
h2 {
	margin: 1rem;
	text-decoration: underline;
}
p {
	margin: 1rem;
}
```
